# Built with .ai Showcase

The fastest way for `.ai` to spread is proof.

Submit a pull request with:

- a short project description
- a sanitized `.ai/build-report.md`
- screenshots or a demo link when public
- the agent/tool used
- the `.ai` pack or workflow used

```markdown
## Project name

- Repo or demo:
- Built by:
- Agent/tool:
- Pack/workflow:
- Build report:
- What changed:

```
