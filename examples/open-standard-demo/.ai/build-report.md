# Built with .ai

## Project

.ai Open Standard Demo

## Idea

Reposition `.ai/` from a markdown method into a standard every agent and IDE
can discover, validate, and extend.

## Active Specs

- Open Standard v0.1

## Outcome

The branch adds `spec.ai.md`, `.ai/manifest.json`, schemas, registry metadata,
CLI conformance checks, helper skills, governance, and a shareable build report.

## Lessons

- The viral loop should create proof artifacts automatically.
- The manifest makes `.ai` usable by tools without sacrificing markdown as the
  source of truth.
